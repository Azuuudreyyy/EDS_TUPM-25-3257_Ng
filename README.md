# EDS_7_N - Engineering Data Systems Pipeline

## Topic: AUT-03: GPS Drift Anomalies
**Assigned Pillar:** Pillar 7: Autonomous Systems & Robotics

This repository contains the Python pipeline for analyzing UAV sensor telemetry to detect GPS anomalies, built for the Computer Programming 1 Final Project.

## Project Structure
- `main.py`: The core OOP data pipeline script.
- `data/`: Directory for input datasets (`dataset_original.csv` and `dataset_cleaned.csv`).
- `outputs/`: Directory for generated graphs and animations.
- `requirements.txt`: Python package dependencies.

## Instructions
1. Place the dataset in the `data/` folder and name it `dataset_original.csv`.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the pipeline: `python main.py`
